塔攻略

遊び方
1. このフォルダを好きな場所に置きます。
2. index.html をブラウザで開きます。
3. 開始ボタンを押すとゲームが始まります。
4. うまく開けない場合は local-server.js を node で起動して、表示されたURLを開きます。

操作
- 移動: WASD / 矢印キー
- 射撃: 左クリック長押しでフルオート
- オート射撃切り替え: F
- リロード: R
- 爆弾: 強化で取得後に Q
- 強斬撃: 強化で取得後に E
- ブリンク: 強化で取得後に Space
- プレイ中に右上の「スタートへ」からスタート画面に戻れます。

メモ
- HTML / CSS / JavaScript だけで動くローカルゲームです。
- スタート画面で難易度を選べます。イージーは敵が弱く報酬が多め、ハード/ゲーマーは敵が強く経験値倍率が高めです。
- エフェクト量を「軽量」にすると爆発やリング演出を減らして処理落ちしにくくします。
- ボス練習では撃破済みボスに挑戦できます。通常プレイでボスを倒すと、その階層が練習に追加されます。
- ボス練習では通常スキルLv、レア/進化/赤スキル、ボスHP、基礎HP、基礎攻撃力、火力突破Lv、生命爆発Lvを調整できます。
- 火力突破Lvと生命爆発Lvは本編と同じく片方だけ有効です。
- 「最後の通常プレイ状態をコピー」で直前の通常プレイの強化構成をボス練習へ反映できます。
- 「スキル構成リセット」はボス練習で選んだスキル構成を初期化します。階層解放状況はそのまま残ります。
- 「銃改造」を取ると、連射速度強化/高速装填/貫通弾/発射数強化が出るようになります。
- 銃改造Lv5以降は進化スキル「魔改造」が開放され、拡大収縮/ヒットストップ/レーザービームが出るようになります。
- 拡大収縮は通常弾を大きくし、追加弾の散らばりを小さくする上位レアスキルです。
- ヒットストップは通常弾命中時、同じ敵に0.2秒ごとに0.02秒の停止を与えます。
- レーザービームは赤スキルで、2.4秒間、銃の連射・装填・弾速・貫通を大幅強化し、Lvごとに再発動間隔が短くなります。
- 爆弾強化と斬撃強化はLv5になると派生強化が開放されます。
- 爆弾Lv5以降は進化スキル「爆発連鎖」で威力を伸ばし、倒した敵も低確率で爆発するようになります。
- 斬撃Lv5以降は進化スキル「波動斬撃」で扇型の波動を発射し、火力・射程・貫通力を強化できます。
- 「技研鑽」はレアスキルで、爆弾と斬撃のクールタイムを短縮します。
- 雷弾はLv5までで、Lvごとに連鎖数が増え、クールタイムが短くなります。
- 雷弾Lv5以降は進化スキル「避雷印」が開放され、雷を受けた敵の被ダメージ倍率を上げます。
- 避雷印Lv5以降は赤色スキル「雷域」が開放され、自分の周囲の敵へ1秒ごとに避雷印を付けます。
- レアスキル「回復装甲」はHPに応じたシールドを獲得し、3秒間被弾しないと自動回復します。
- 回復装甲Lv5以降は進化スキル「自然力上昇」が開放され、HP自動回復と被ダメージ軽減を得ます。
- 3層ボス撃破後は「火力突破」か「生命爆発」のどちらかを選びます。片方を選ぶともう片方は消滅します。
- 火力突破はプレイヤーの主力攻撃系火力倍率を上げます。
- 生命爆発は基礎HPをコピーして最大HP倍率を上げます。
- レア強化と進化スキルは基本Lv5まで伸びます。
- 4層ボスは反射弾を撃つ召喚敵を呼び、5層ボスは時間経過で復活するシールドを持ちます。
- 5層ボスはHP半分で試練を開始し、黄色敵を倒しきれないとボスが回復します。
- ボスの丸形/縦型攻撃はステージが上がるほど同時発生数が増えます。
- ボス斬撃は広い扇形に、棒状の斬撃が右から左へ振り払われます。
- 被弾するとノックバックし、通常敵とは重なりにくくなっています。
- 被弾時は画面端が赤く光ります。
- ボス条件を満たすと敵の出現が止まり、散らばった経験値を吸収して、ボス扉が出現します。
- 右下のミニマップには敵、ボス扉、宝箱が表示され、遠い宝箱は画面端に方向アイコンが出ます。
- 大きい青敵は斬撃前にプレイヤー近くへブリンクします。
- 通常敵には大型版が出現し、色ごとの特徴がさらに強化されます。
- 友達に送るときは survival フォルダごとzipにまとめて送ってください。受け取った側は解凍して index.html を開けば遊べます。
